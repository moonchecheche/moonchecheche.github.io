# 개인정보 처리방침

**어제오늘날씨** (이하 "앱")는 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다.

## 1. 수집하는 정보

### 위치 정보

- 앱은 날씨 정보를 제공하기 위해 사용자의 **현재 위치(GPS)**를 수집합니다.
- 위치 정보는 날씨 API 요청에만 사용되며, 서버에 저장되지 않습니다.

### 수집하지 않는 정보

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 사용자 계정 정보
- 기기 고유 식별자

## 2. 정보의 이용 목적

수집된 위치 정보는 다음 목적으로만 사용됩니다:

- 현재 위치 기반 날씨 정보 제공
- 어제와 오늘의 날씨 비교 정보 제공

## 3. 정보의 보관 및 삭제

- 위치 정보는 기기 내에서만 임시로 처리되며, 외부 서버에 저장되지 않습니다.
- 앱을 삭제하면 모든 로컬 데이터가 자동으로 삭제됩니다.

## 4. 제3자 제공

- 날씨 정보를 가져오기 위해 위치 좌표가 날씨 API 서비스로 전송됩니다.
- 그 외 어떠한 제3자에게도 개인정보를 제공하지 않습니다.

## 5. 사용자의 권리

- 사용자는 언제든지 기기 설정에서 위치 권한을 거부하거나 철회할 수 있습니다.
- 위치 권한을 거부해도 앱을 사용할 수 있으며, 이 경우 기본 위치(서울)의 날씨가 표시됩니다.

## 6. 광고

- 앱에는 Google AdMob 광고가 포함될 수 있습니다.
- 광고 서비스는 자체적인 개인정보 처리방침을 따릅니다.

## 7. 아동의 개인정보

- 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 8. 개인정보 처리방침의 변경

- 본 방침이 변경될 경우, 앱 내 공지 또는 업데이트를 통해 안내합니다.
